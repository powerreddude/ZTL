## Layout

Following true Zettlekasten style all notes are added in `./notes` and no 
nesting or hierarchy is allowed under any circumstances. *However* this only 
applies to notes and the following directories were added to keep maintenance 
and usability high. Thus the following layout is used for the root of this 
repository: 

| Path        | Use                                                  |
| ----------- | ---------------------------------------------------- |
| /notes/     | Zettle \[Note/s\]                                    |
| /scripts/   | Scripts for maintenance or usability                 |
| /templates/ | Templates for generating notes                       |
| /tags/      | Generated links all [Tags](#tags)                    |
| /sorted/    | Generated letter directories with links to all notes |
| /.config    | Configuration file                                   |
